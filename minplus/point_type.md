Returns the type of the point for given vertices. The definition of type was the one used in [^5] and [^6].

Parameters
: `point : Collection`: the point for which the type is calculated.
: `vertices : Collection`: the vertices in the context of which the type will be calculated.
: `indexing_start : int`: the initial index for the items of the type. Defaults to 0.

Returns
: `result : list`: the type of the point in the context of the vertices.
