An implementation of a tropical hyperplane.

Constructor parameters
: `*coefficients : float`: a collection of coefficients of the hyperplane.

#### Hyperplane.get_value(point)

Calculates the value which the hyperplane achieves at a given point.

Parameters
: `point : Collection`: the point for which the value will be calculated.

Returns
: `result : float`: the value which the hyperplane achieves at the given point.

#### Hyperplane.get_apex()

Returns
: `results : tuple`: the point which is the apex of the hyperplane.
